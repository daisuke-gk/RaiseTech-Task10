# CRUD処理を実装したAPI
## 概要
名前とメールアドレスの一覧のAPIを作成しました。

### APIの内容
* Read処理

  * 全件検索及び部分一致検索(クエリパラメータ)の実装
  * id検索(パスパラメータ検索)の実装
  * id検索(パスパラメータ検索)の例外処理(NotFoundException)
